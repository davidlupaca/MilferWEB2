
MILFER - Web final (Formulario + Dashboard)

Archivos incluidos:
- index.html (sitio único, navegación, formulario y dashboard)
- styles.css (estilos)
- app.js (lógica principal que conecta con tu Apps Script)
- dashboard.js (script separado para dashboard—opcional)
- README.md (este archivo)

Instrucciones rápidas:
1. Sube estos archivos a un hosting (GitHub Pages es gratuito y recomendado).
2. Asegúrate de que tu Apps Script esté desplegado como "Web app" y acceso "Anyone".
3. Si tu Apps Script devuelve JSON con `rows` como arreglo con headers, la web lo manejará. Si devuelve otro formato, dime y adapto.
4. Abre index.html en el navegador (o la URL de GitHub Pages) y prueba registrar un movimiento.
5. El dashboard mostrará resumen y gráficos automáticos.

Si quieres, publico esto en un repo por ti (necesitaría permiso para crear el repo) o te doy pasos para publicarlo en GitHub Pages.
